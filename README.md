# Infosys Skill Build Program - AI Learning Track

Welcome to the **Infosys Skill Build Program** repository. This program includes a learning track focused on foundational AI concepts, aimed at building a strong understanding of artificial intelligence and its practical applications.

The Infosys Skill Build Program aims to provide freshers with essential skills in AI, data science, and machine learning. This repository contains resources, projects, and assignments to help understand and implement AI concepts practically.

## Daily AI Breakdown

### Day 1: Introduction to AI
- **Topics Covered**: What is AI, history and evolution, types of AI (Narrow AI, General AI, Super AI).
- **Objective**: Understand the basics and different branches of AI.
- **Assignments**:
  - Write a summary on the types of AI and their characteristics.
  - Research real-world examples of AI applications.

### Day 2: Machine Learning Fundamentals
- **Topics Covered**: Introduction to machine learning, types of machine learning (supervised, unsupervised, reinforcement).
- **Objective**: Learn the foundational concepts in machine learning.
- **Assignments**:
  - Create a simple linear regression model in Python.
  - Identify examples for each type of machine learning.

### Day 3: Data Preparation and Preprocessing
- **Topics Covered**: Data collection, cleaning, and preprocessing techniques (normalization, handling missing values).
- **Objective**: Learn to prepare data for AI and machine learning models.
- **Assignments**:
  - Load a dataset and apply preprocessing steps.
  - Practice data cleaning techniques.

### Day 4: Neural Networks and Deep Learning Basics
- **Topics Covered**: Introduction to neural networks, structure of a neural network, basics of deep learning.
- **Objective**: Understand the architecture of neural networks and how they work.
- **Assignments**:
  - Create a simple neural network model.
  - Study the impact of different activation functions.

## Technologies Used

- **Programming Languages**: Python
- **Libraries**: NumPy, pandas, scikit-learn, TensorFlow or PyTorch (for neural networks)
- **Tools**: Git, GitHub, Jupyter Notebook, Visual Studio Code
